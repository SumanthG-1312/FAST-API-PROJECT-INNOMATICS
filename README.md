# Hotel Management System (FastAPI)

This is a clean and simple project to manage hotel activities like booking rooms, checking availability, and viewing summary reports. It is built using FastAPI and Python for quick execution and easy understanding.

### Features
*   View all available rooms and their details.
*   Book rooms with specific meal plans.
*   Check-in and check-out management.
*   Filter and sort rooms by price or floor.

It's a complete solution for anyone starting with API development in Python.
